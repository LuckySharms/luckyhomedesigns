Privacy Policy

Lucky Home Designs — Last updated March 1, 2026

Overview

Lucky Home Designs ("the Application") is a personal content management tool used to publish and manage pins on a single Pinterest Business account. The Application is operated solely by the account owner and is not a multi-user platform. It does not provide services to third parties, nor does it collect personal information from external users.

Data Accessed Through Pinterest

The Application connects to the Pinterest API (v5) using OAuth 2.0 authorization. Through this connection, the Application accesses the following data belonging to the account owner:

  • Account profile information (username, account type)
  • Boards (names, descriptions, privacy settings)
  • Pins (titles, descriptions, images, destination links, analytics)
  • Pin-level and account-level analytics (impressions, saves, clicks)

This data is accessed solely to manage the account owner's own Pinterest content. No data from other Pinterest users is accessed, collected, or stored.

Data Collection

The Application does not collect personal information from any visitors or third parties. No cookies, tracking pixels, or analytics scripts are used on this website. The only data processed by the Application is the account owner's own Pinterest content and API credentials.

Data Storage and Retention

API access tokens are stored securely on a private server controlled by the account owner. These tokens are refreshed automatically per Pinterest's OAuth 2.0 specifications (30-day access token, 365-day refresh token) and are not shared with any third party.

Content created through the Application (pin images, titles, descriptions) is uploaded to Pinterest via the API and to a private content delivery network (CDN) for image hosting. No Pinterest user data is cached or retained beyond what is necessary for the Application's content management functions.

Third-Party Services

The Application interacts with the following third-party services in the course of normal operation:

  • Pinterest API (v5): Used to publish pins, manage boards, and retrieve performance analytics for the account owner's content.
  • DigitalOcean Spaces: Used for hosting pin images via a private CDN. No personal data is stored on this service.
  • Amazon Associates: Affiliate links within published pins may direct to Amazon product pages. Amazon's own privacy policy governs interactions on their platform.

No data obtained from the Pinterest API is shared with, sold to, or disclosed to any third party, advertising service, or data broker.

Data Sharing

The Application does not sell, rent, trade, or otherwise share any data with third parties. Data accessed through the Pinterest API is used exclusively to manage the account owner's own content.

Children's Privacy

The Application is not directed at children under the age of 13 and does not knowingly collect information from children.

Changes to This Policy

This privacy policy may be updated from time to time. Any changes will be reflected on this page with an updated revision date.
